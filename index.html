About Me

My name is Samuel Radford. This resume page was created as a portfolio project in my Web Development Boot Camp. It is meant to show off my education, skills, experience, and projects. I began studying Computer Science my junior year of high school, where we learned using Pascal and C++. I continued on to study Computer Science at Appalachian State University where I learned skills in Java, C, HTML, CSS, JavaScript, JQuery, and MySQL. While in school I taught myself development for Android. I received a degree in Computer Science with a minor in Mathematics from Appalachian State Univeristy. Next, I attended a boot camp which focused on HTML, CSS, JavaScript and jQuery, MySQL, and C#. After the boot camp, I started a job as a full-stack web developer for AT&T where we worked on a project that brought many apis together to create different workflow systems that AT&T technicians used to do their work. We used a Perl back end along with the typical HTML, CSS, JavaScript front end using Ember.JS as a framework. On my own time I am currently learning Python and Angular as I build a web application to create and track tasks for a project which will be used to keep my organized while I do personal development work.
